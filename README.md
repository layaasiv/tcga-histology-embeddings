# Generating Embeddings for Whole Slide Images of Cancer Tissues using Prov-GigaPath

This repo contains my contributions to a larger [project](https://www.biorxiv.org/content/10.1101/2025.05.14.653900v1.full#sec-9) studying signal preservation in integrated heterogenous embedding vectors.

Whole slide images were sourced from TCGA. These were the cancers of interest; all available WSIs were downloaded:
* (STAD) Stomach adenocarcinoma
* (COAD) Colon adenocarcinoma
* (LAML) Acute Myeloid Leukemia
* (BRCA) Breast invasive carcinoma
* (THCA) Thyroid carcinoma
* (BLCA) Bladder Urothelial Carcinoma


| File | Description |
| ---- | ----------- |
| environment.yml | Packages needed to create conda environment for this work. |
| download_annotations.py | Script used to download data from TCGA. |
| tcga-gp-final.py | OOP script used to process, quality filter, and extract and save image tile-level embeddings generated by the pathology foundation model, Prov-GigaPath. |
